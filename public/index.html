How to Use

  1. Open ATEM Software Control, go to File > Save As, and export your switcher state to an .xml file.
  2. Drag and drop your .xml file into the drop zone above.
  3. Use the tabs on the left to quickly configure your Labels, Multiviews, Auxiliaries, and Audio mapping.
  4. Click Export XML in the top right corner.
  5. In ATEM Software Control, go to File > Restore and select your newly downloaded file!
